C program vs. Java Programs
=========================================================================
=====================================================
-C programs are fast and the prerequisite is low
-java is slow, requires OS + Java virtural machine. Not very good for
small devices
C program directly manipulate memory
=========================================================================
=====================================================
int main()
{
int a = 0;
int * p = &a;
return 0;
printf("%ld", (long)(*p));
p = (int*)malloc(100*sizeof(int));
}
=========================================================================
=====================================================
int * p = &a;
declare a pointer-type of variable named p, and initialize to the address
of variable a.
in a declaration, "*" stands for pointer type.
After words, "*" means "de-refrence" retrieving the value at the memory
location, using the syntax *p.
